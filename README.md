# HSPDS Optimized Dashboard

This is a smaller GitHub-friendly version of the original single-file dashboard.

## Files

- `index.html` — dashboard app
- `assets/` — compressed WebP reference images

## How to use

Upload the full folder contents to GitHub, not just `index.html`.

For GitHub Pages:
1. Put these files in the repository root.
2. Enable GitHub Pages from the repository settings.
3. Set the source to the main branch/root folder.

## Note

The model/export functions are conceptual mesh exports, not manufacturing-certified CAD.
